После установки Ubuntu  откройте  программу из Пуска, или наберите
bash 
либо в командной строке Win, либо в адресной строке проводника.

При первом запуске (после продолжительного ожидания) будет предложено
придумать имя пользователя и пароль. Эти данные придется вводить при 
установке (удалении) новых пакетов.

Для работы необходимо подкачать пакеты, это можно сделать командой:
sudo apt install build-essential
Будет установлен компилятор, редактор vim и некоторые другие пакеты. 

Для обновления пакетов (которое нужно периодически делать)
надо ввести команды

sudo apt update
(это получение сведений об обновлениях)
sudo apt full-upgrade
(сами обновления)

Для того, чтобы настроить терминал,  нажмите мышкой в его левом 
верхнем углу и выберете Свойства. Можно выбрать размер шрифта 
(на вкладке шрифт). На вкладке настройки->Правка  отметьте галочкой:
используйте сочетания клавиш CTRL+SHIFT+C/CTRL+SHIFT+V. Это позволит 
в редакторе Win  отметить текст мышью или клавишами  (Shift + стрелка), 
скопировать его в буфер обмена и потом вставить в bash. Привычные 
сочетания Ctrl+C, Ctrl+V здесь не работают.

Для выхода из bash наберите 
exit
